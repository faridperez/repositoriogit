Comandos más Utilizados de Git

Git ofrece una amplia variedad de comandos que nos facilitan trabajar de manera eficiente a lo largo de un proyecto, estos comandos nos permiten tener un mayor control. Los comandos mas utilizados de Git son:

1.Clona un repositorio

git clone URL del repositorio
Con este comando copiamos el repositorio remoto en la maquina local, permitiendo trabajar en él.

2.Bajar los cambios

git pull nombre-de-la-rama
Con este comando bajaremos todos los cambios subidos al repositorio.

3.Crear una rama

git checkout -b nombre-de-la-rama git checkout nombre-de-la-rama
Con el primer comando crearemos ua rama nueva con la cual trabajaremos y el segundo cambiaremos a la rama que deseamos trabajar o simplemente ver sus cambios.
git bransh nombre-de-la-rama git branch -m antiguo-nombre nuevo-nombre git branch -d nombre-de-la-rama
estos comandos crean, renombran y borran una rama respectivamente.

4.Agregar y Confirmar cambios

git status git add . git commit -m "descripción del cambio(s)
Los comandos anteriores se pueden ver los cambios realizados en la rama, agregar los cambios a los archivos y confirmar los cambios realizados.

5.enviar cambios al repositorio

git push origin nombre-de-la-rama
Con este comando se envian los cambios realizados al repositorio.

6.Historial de cambios

git log
Despliega una lista de los cambios realizados en el repositorio con la fecha y hora respectiva de modificacion.

7.Sacar cambios

git status git restore
Se usa para deshacer cambios, git status se ejecutara si queremos cambiar un archivo en especifico, si deseamos cambiar el proyecto completo ejecutaremos git restore.

8.Guardar temporalmente

git stash save "Descripcion opcional" git stash list
Se usa para guardar temporalmente los cambios que no han sido subidos al repositorio, el segundo comando muestra la lista de todos los stashes creados y sus respectivas descripciones.
